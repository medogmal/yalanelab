## 📐 القواعد التقنية الثابتة

```
Framework:   Next.js 16 (App Router)
Language:    TypeScript (strict)
Styling:     Tailwind CSS v4 + Custom CSS vars
Animations:  Framer Motion
State:       Zustand (global) + useState (local)
DB:          Prisma + SQLite (dev) → PostgreSQL (prod)
Auth:        NextAuth.js v4
Realtime:    Socket.IO
Icons:       Lucide React
Fonts:       Cairo + Noto Naskh Arabic
```

### قواعد التصميم الثابتة
- **RTL دائماً** — العربية أولاً
- **Mobile First** — الشاشة الصغيرة أهم
- **لا عناصر بدائية** — كل حاجة SVG أو صور أو CSS احترافي
- **الفريم الثقافي** على كل طاولة لعبة
- **لوجو يالا نلعب** على الطاولة (watermark خفيف)
- **3 موودات** — كل مود يغير كل حاجة
